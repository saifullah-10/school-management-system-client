## Features

### General Features
- **Role-Based Authentication**: Secure login with role-based access for students, teachers, and administrators.
- **Responsive Design**: Fully responsive interface for access on any device.
- **Secure Login Form**: Ensures that user authentication is handled securely, protecting user data and providing a safe login experience and also allows new users to create an account, making it easy to get started with the platform and gain access to its features.
- **Course Listings**: Displays a list of available courses with details and user ratings. This helps users make informed decisions about which courses to enroll in based on comprehensive information and feedback from others.

### Student Features
- **Course Enrollment**: Enroll in available courses directly from the dashboard.
- **Personalized Class Schedule**: View and manage class schedules tailored to individual timetables.
- **Attendance Tracking**: Monitor personal attendance records and patterns.
- **Exam Management**: Access details of upcoming exams and view past exam results.
- **Assignments Tracking**: Track weekly assignments, their due dates, and submission status.
- **Notice Board**: View all relevant notices and announcements related to coursework and events.
- **Profile Management**: Update personal details and manage account settings.

### Teacher Features
- **Class Schedule Management**: Create and manage class schedules, including timings and locations.
- **Student Attendance Tracking**: Record and review student attendance data.
- **Assignment Management**: Create and assign assignments, track submissions and grades.
- **Gradebook**: Manage and update student grades, view grade distribution.
- **Exam Scheduling**: Set up and manage exam schedules, including creating and grading exams.
- **Notice Creation**: Post and manage notices for students and staff.
- **Student Insights**: Access analytics on student performance and attendance.

### Admin Features
- **Comprehensive Dashboard**: Overview of key metrics including student enrollment, staff details, and financial data.
- **User Management**: Add, update, or remove users and manage roles.
- **Course and Exam Management**: Create, update, and oversee courses and exams.
- **Notice Administration**: Create, update, and manage notices and announcements.
- **Financial Tracking**: Monitor and manage school expenses and revenues.
- **Data Backup and Restore**: Backup and restore system data to prevent loss.
- **Report Generation**: Generate detailed reports on various aspects of the school management.

## Pages Overview

### Home Page
The Home Page provides an overview of the application and access to key features.

#### Features
- **Prominent Call-to-Action**: Buttons to sign up or log in.
- **Service Highlights**: Overview of core services and features.
- **User Testimonials**: Feedback from users about the application.
- **Navigation Links**: Direct links to major sections of the site.

### About Us
The About Us page details the institution’s mission and key team members.

#### Features
- **Mission Statement**: Overview of the institution's goals and values.
- **Team Introduction**: Profiles of key faculty or administrative staff.
- **Institutional History**: Background information about the school.

### Contact Us
The Contact Us page facilitates communication between users and administration.

#### Features
- **Contact Form**: For submitting queries and feedback.
- **Contact Information**: Address, phone number, and email details.
- **Social Media Links**: Links to the school’s social media profiles.

### Class Routine
The Class Routine page displays students' class schedules.

#### Features
- **Class Schedule Table**: Detailed table of class timings and locations.
- **Instructor Information**: Details about the instructors for each class.
- **Schedule Download**: Option to download the class schedule in various formats.

### Attendance
The Attendance page tracks student attendance records.

#### Features
- **Attendance Table**: Table showing attendance records with filters.
- **Attendance Summary**: Graphical representation of attendance trends.
- **Export Attendance Data**: Option to export attendance records.

### Exam Page
The Exam Page provides information on exams and assignments.

#### Features
- **Upcoming Exams**: List of upcoming exams with details.
- **Weekly Assignments**: Overview of assignments including due dates and statuses.
- **Exam History**: Historical data on past exams and results.

### Notice Page
The Notice Page allows for the creation and viewing of notices.

#### Features
- **Notice Creation**: Form to create new notices with title, details, and date.
- **Notice Board**: Display of all notices with titles, dates, and brief descriptions.
- **Search Notices**: Functionality to search through notices.

### Student Dashboard
The Student Dashboard provides a comprehensive view of academic activities.

#### Features
- **Class Routine**: Display of the student's class schedule.
- **Attendance Overview**: Summary of personal attendance records.
- **Course Enrollment**: Interface for enrolling in new courses.
- **Exam and Assignment Tracking**: Overview of upcoming exams and current assignments.
- **Notice Board**: Access to all relevant notices and updates.

### Admin Dashboard
The Admin Dashboard offers tools for managing the school’s operations.

#### Features
- **Statistics Overview**: Key metrics such as student numbers and financial summaries.
- **User Management**: Tools to manage user roles and accounts.
- **Course and Exam Administration**: Options for creating and managing courses and exams.
- **Notice Management**: Tools for creating and administering notices.
- **Financial Tracking**: Overview of financial data including expenses and revenues.
- **Report Generation**: Ability to generate detailed reports on various school operations.

### Teacher Dashboard
The Teacher Dashboard provides teachers with essential tools for managing their classes and students.

#### Features
- **Class Schedule Management**: Tools to create and update class schedules.
- **Student Attendance Tracking**: Record and monitor student attendance.
- **Assignment and Exam Management**: Manage and grade assignments and exams.
- **Notice Board**: Post and view notices relevant to teachers and students.
- **Student Performance Insights**: Access detailed reports on student performance.

## Usage
- **Students**: Manage courses, track attendance, view exam schedules, and stay updated with notices.
- **Teachers**: Organize class schedules, track attendance, assign grades, and communicate with students.
- **Admins**: Oversee the system, manage users and courses, track finances, and generate reports.

## Benefits
- **Efficiency**: Streamlines academic and administrative processes.
- **Communication**: Keeps all users informed with timely updates and notices.
- **Management**: Simplifies the management of courses, attendance, exams, and other academic activities.

## Installation
To set up the application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/saifullah-10/school-management-system-client.git
   ```
2. Navigate to the project directory:
   ```bash
   cd school-management-system
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm run dev
   ```
